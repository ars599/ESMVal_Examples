Performance metrics for essential climate variables in CMIP5

Recipe for plotting the performance metrics for the CMIP5 datasets, including the standard ECVs as in Gleckler et al., and some additional variables (like ozone, sea-ice, aerosol...)

Authors

  • Franziska Frank (DLR, Germany; None)

Maintainers

Projects

  • EU H2020 project CRESCENDO
  • Copernicus Climate Change Service 34a Lot 2 (MAGIC) project
  • ESA CMUG
